```sh
npm i
npm run dev
```

### Prerequisites

Ensure you have the following software installed:

- **Node.js:** Version 20 or higher
- **React.js:** Version 19 or higher
- **Tailwind CSS:** Version 4 or higher
- **Shadcn UI:** Version 0.9.5 or higher
